# Moonscribe Specification Sheet

## Purpose

Moonscribe will be a program used to track combat initiative in Dungeons and Dragons 5th edition. As a part of that function it will contain a database of all 5th edition SRD creatures and their statblocks. This database will be editable by the user to store custom creatures and modify existing creatures as they desire. There will be fucntionality to store player data, create and save encounters ahead of time, create encounters on the fly, and edit saved encounters ahead of time or on the fly. Specific to tracking initiative, Moonscribe will be able to generate initiative for combatants as well as take user input of combatant initiative. It will provide a method of tracking combatant hitpoints and a method to track which initiative turn is active as well as displaying the general order.

## Scope

1. Combat Features
   1. Assign initiative automatically or based on user input
   2. Provide creature stat blocks
   3. Track creature hitpoints
   4. Track active initiative turn
   5. Modify combatants and initiatives in active combat encounters
2. Creature Database Features
   1. Store all 5th edition SRD creatures and their statblocks
   2. All statblocks will be searchable based on name
   3. All statblocks will be filterable and sortable based on source material, name, type, size, and challenge rating
   4. New creatures will be able to be created with editable fields for all standard statblock features
   5. Existing creatures will be able to be modified
3. Player Database Features
   1. Players will be able to be added to the database to be used in combat scenarios
   2. Player data may include:
      1. Name
      2. Armor Class
      3. Player Class
      4. Original Campaign
4. Out of Scope
   1. 5.5 edtion of Dungeons and Dragons (revisit at a later date)

## Audience

The target audience is Game Masters of Dungeons and Dragons 5th Edition.

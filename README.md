# Moonscribe_lvgl

Moonscribe_lvgl is a graphical version of [clidnbatter](https://github.com/rihi52/clidnbatter) written completely in C, using lvgl as the graphics library. This is in the very early stages. It will eventually consist of all the features contained in [myClayoutTesting](https://github.com/rihi52/myClayoutTesting), which is a project created to test out using [Clay](https://github.com/nicbarker/clay).
